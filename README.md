# babel-preset-niksy

[![Build Status][ci-img]][ci]

[Babel][babel] preset for my projects.

## Install

```sh
npm install babel-preset-niksy --save
```

## Usage

Add this config to your `.babelrc`:

```json
{
	"presets": [
		"niksy"
	]
}
```
